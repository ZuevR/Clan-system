CONFIGURATION
-------------

### Database

Create and edit the file `config/db_params.php` with real data, for example:

```php
return [
  'host' => 'mysql:host=localhost',
  'dbname' => 'yourdbname',
  'user' => 'root',
  'password' => '****',
];
```
